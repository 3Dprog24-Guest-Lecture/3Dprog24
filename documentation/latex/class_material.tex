\doxysubsection{Detailed Description}
Manages material properties and textures for rendering. 

\begin{DoxyNote}{Note}
This class cannot be copied or assigned due to the deletion of the copy constructor and copy assignment operator, ensuring unique instances for the use of the lightweight pattern. 
\end{DoxyNote}
