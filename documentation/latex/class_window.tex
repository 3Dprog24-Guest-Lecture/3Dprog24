\doxysubsection{Detailed Description}
Represents a window in the application, managing its lifecycle, events, and rendering context. 

The \mbox{\hyperlink{class_window}{Window}} class encapsulates a G\+L\+FW window and its associated context. It is responsible for initializing the window, handling input events through callbacks, and managing the render loop. The class also integrates with a \mbox{\hyperlink{class_scene}{Scene}} object to update and render the content within the window.
