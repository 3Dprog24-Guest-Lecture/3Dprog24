\doxysubsection{Detailed Description}
Controls actor movement based on user input. 

\mbox{\hyperlink{class_actor_controller}{Actor\+Controller}} is responsible for handling user inputs. It is designed to work with a \mbox{\hyperlink{class_actor}{Actor}} to update its transformation based on the input.
