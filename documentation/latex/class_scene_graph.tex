\doxysubsection{Detailed Description}
Represents the root node of a scene\textquotesingle{}s hierarchical structure. 

\mbox{\hyperlink{class_scene_graph}{Scene\+Graph}} extends the \mbox{\hyperlink{class_actor}{Actor}} class to act as the root node in a scene\textquotesingle{}s hierarchy, allowing for the organization and management of a collection of actors within a scene.

\begin{DoxyNote}{Note}
This class currently does not implement a bounding volume hierarchy (B\+VH) for spatial partitioning, which can be used for efficient collision detection and culling. 
\end{DoxyNote}
