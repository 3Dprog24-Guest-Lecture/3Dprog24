\doxysubsection{Detailed Description}
Represents an actor in the scene that holds a mesh. 

\mbox{\hyperlink{class_mesh_actor}{Mesh\+Actor}} is a specialized type of actor designed to hold a mesh and render it. It inherits from the \mbox{\hyperlink{class_actor}{Actor}} class to integrate into the scene graph and implements the \mbox{\hyperlink{class_i_render}{I\+Render}} interface to provide rendering functionality. This allows the \mbox{\hyperlink{class_mesh_actor}{Mesh\+Actor}} to be managed within the scene graph while also having the capability to render a specific mesh using a shader.
