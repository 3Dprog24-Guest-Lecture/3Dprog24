\doxysubsection{Detailed Description}
Represents an actor in the scene that holds a mesh. 

\doxylink{class_mesh_actor}{Mesh\+Actor} is a specialized type of actor designed to hold a mesh and render it. It inherits from the \doxylink{class_actor}{Actor} class to integrate into the scene graph and implements the \doxylink{class_i_render}{IRender} interface to provide rendering functionality. This allows the \doxylink{class_mesh_actor}{Mesh\+Actor} to be managed within the scene graph while also having the capability to render a specific mesh using a shader.
