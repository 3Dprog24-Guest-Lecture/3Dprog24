\doxysubsection{Detailed Description}
Interface for input controllers. 

The \doxylink{class_i_controller}{IController} interface defines the structure for input handling in the application. It includes methods for processing mouse movement, mouse scrolling, mouse button actions, and keyboard actions. A controllable object can inherit from this, only one active controller will proccess the inputs. This can be used for camera, player or whatever object that needs to be controlled.
