\doxysubsection{Detailed Description}
Controls camera movement and orientation based on user input. 

\doxylink{class_camera_controller}{Camera\+Controller} is responsible for handling user inputs. It is designed to work with a \doxylink{class_camera_actor}{Camera\+Actor} to update its transformation based on the input.
