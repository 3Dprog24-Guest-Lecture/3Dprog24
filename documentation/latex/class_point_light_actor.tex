\doxysubsection{Detailed Description}
A concrete actor class representing a point light source in a 3D scene. 

Inherits from both \mbox{\hyperlink{class_actor}{Actor}} and \mbox{\hyperlink{class_point_light}{Point\+Light}} to implement a point light that can be positioned and moved within the scene. It provides the implementation of Get\+Position to return the global position of the light actor.
